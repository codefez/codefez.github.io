try and except

The try/except blocks test out the code you put under the "try" and will display a message that you choose under "except". After using "except", you can also add a specific error that will print out the specific message you wish for it to print out. These errors are known as python exceptions. 




 try:
   print(x)
 except:
   print("There was a problem in your code.")

          
There was a problem in your code.
