sleep

To be able to use sleep you need to from time import sleep. Then, using the function sleep(1), you can specify the amount of seconds within the parentheses (). 


  from time import sleep
  
  # Waits 3 seconds
  sleep(1)
  print("hello")
  # Waits 5 seconds
  sleep(5)
  print("world")
