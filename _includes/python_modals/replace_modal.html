Replace function

The replace() function can be used to replace a previous variable. In this example I also use sleep function to model it. You can find it in the cheatsheet. 




  from time import sleep
  
  original = "Hello, World!"
  print(original) # Output: Hello World
  sleep(1)
  original.replace("World", "Hello")
  print(original) # Output: World Hello
