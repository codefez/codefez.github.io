Indexing Characters

Strings are indexed in programming languages. This means that each character in a string has a position number (index)
Indexing starts at 0, so the first character of a string is at index 0.
You can access individual characters in a string by specifying their index number.

Example 1

    string = "Codefez"
    char1 = string[4]
    char2 = string[5]
    char3 = string[6]
    print(f"{char1}{char2}{char3}.")
