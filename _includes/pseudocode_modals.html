Input and Output

We can use the input() function to get user input and the print() function to display output.

name = input("What is your name? ")
print("Hello, " + name + "!")

The above code will prompt the user to enter their name and then greet them with a personalized message. For example, if the user enters "Alice", the output will be:

Hello, Alice!

Input and Casting

We can cast the result of input() to a different datatype if needed.

number = int(input("Enter a number to work out the square: "))
print("The square of " + str(number) + " is " + str(number * number) + ".")

The above code will prompt the user to enter a number and then display its square. For example, if the user enters "5", the output will be:

The square of 5 is 25.

By default, the input() function returns a string, so we need to cast it to an integer using int() before performing the calculation.

String Indexing

We can use indexing to access individual characters in a string. With pseudocode this is done by using the substring method.
The first argument given as a parameter is the starting index, and the second argument is the number of characters to extract beginning from the index position. 

string = "codefez"
print(string.substring(0))
print(string.substring(4,3))

The above code will output the first character of the string "codefez" and then the characters from index 4 to 6. For example, the output will be:

c
fez
Note:The substring method second argument differs from most programming languages such as Python, where the second argument represents the ending index.

for loops

We can use for loops to repeat a block of code a specific number of times.

This is known as a counter-controlled loop because we use it when we know the number of iterations in advance.

for i=1 to 5
    print(i)

The above code will repeat the print statement 5 times. For example, the output will be:

1
2
3
4
5

The variable i is called the loop counter. It is initialized to 1 and incremented by 1 each time the loop executes, taking on the values from 1 to 5 during the execution of the for loop.
It is also known as a local variable because it only exists within the scope of the loop. It cannot be accessed outside of the loop.

while loops

A while loop is condition-controlled. The condition is checked before each loop iteration.

while answer != "Correct"
    answer = input("New answer")
endwhile

This loop continues until the user inputs the string "Correct".

The condition is checked first. If it is false at the start, the loop does not run.

Key points:

  • while ... endwhile defines the loop block.
  • This is used when the number of repetitions is not known in advance.
  • The condition is checked before entering the loop body each time.

do...until loops

A do-until loop is condition-controlled, but it checks the condition after the loop body runs.

do
    answer = input("New answer")
until answer == "Correct"

This loop continues until the user inputs the string "Correct".

The loop runs once before the condition check is carried out.

Key points:

  • do ... until ... defines the loop.
  • This is useful when the loop must execute at least once.
  • The condition is checked at the end of each iteration.

arrays: storing multiple items

An array is a data structure used to store more than one item under one variable name.

scores = [72, 68, 91, 84]
print(scores)

This keeps related values together in a single structure, which makes data easier to process.

  • Use an array when you need to store many values of the same type.
  • Items are stored in a fixed order.
  • Each item can be accessed by its index position.

arrays: indexing items

Each item in an array has an index. The index is used to read or update one specific item.

scores = [72, 68, 91, 84]
print(scores[0])
scores[1] = 70
print(scores[1])
In OCR-style pseudocode, indexing starts at 0, so scores[0] is the first item.

2D arrays

A 2D array stores data in rows and columns (an array of arrays).

seats = [["A1", "A2", "A3"],
         ["B1", "B2", "B3"]]

print(seats[1][2])
seats[row][column] means row index first, then column index.

2D arrays are useful for grids, boards, timetables, and tables of values.

sub-routines: procedures

A procedure is a named block of code that performs a task. It does not return a value.

procedure showMenu()
    print("1. Play")
    print("2. Exit")
endprocedure

showMenu()
  • Use a procedure to avoid repeating common steps.
  • Call the procedure by its name.
  • Procedures improve readability and structure.

sub-routines: functions

A function is a sub-routine that returns a value to the line of code that called it.

function doubleNumber(n)
    return n * 2
endfunction

result = doubleNumber(6)
print(result)
The function call gives back a value, so it can be stored in a variable.

sub-routines: parameters

Parameters allow values to be passed into procedures or functions as arguments.

procedure greet(name, yearGroup)
    print("Hello " + name + " from Year " + yearGroup)
endprocedure

greet("Ava", "10")
  • name and yearGroup are parameters.
  • "Ava" and "10" are arguments.
  • Arguments are matched to parameters by position.

sub-routines: global variables

By default, variables declared inside a subroutine are local — they only exist within that subroutine. Use the global keyword to access and modify a variable that was declared outside.

global score = 0

procedure addPoint()
    global score
    score = score + 1
endprocedure

addPoint()
print(score)
1
  • global score = 0 declares score as a global variable.
  • Inside the procedure, global score tells the subroutine to use the global version.
  • Without the global keyword, score inside the procedure would be a separate local variable.

file handling: reading a file

To read data from a file, open the file, read from it, then close it when you are finished.

myFile = open("user.txt")
line = myFile.readLine()
print(line)
myFile.close()
  • open() opens the file.
  • readLine() reads the next line from the file.
  • close() closes the file when finished.

file handling: writing to a file

We can write data to a file one line at a time using writeLine().

f = open("log.txt")
f.writeLine("Start")
f.writeLine("End")
f.close()
Each call to writeLine() writes one new line to the file.

file handling: looping through a file

A common pattern is to loop through a file until the end is reached.

count = 0
file = open("data.txt")
while NOT file.endOfFile()
    line = file.readLine()
    count = count + 1
endwhile
file.close()
print(count)
  • endOfFile() checks whether there is any data left to read.
  • This pattern is useful for counting, searching, or processing every line.
  • Close the file after the loop has finished.

Case-Switch Selection

A case statement is used when one variable or expression can match several different fixed values.

grade = "B"

switch grade of
    "A": print("Excellent")
    "B": print("Good")
    "C": print("Pass")
    default: print("Keep trying")
endswitch
This would output: Good
  • Use switch ... of when checking one value against many options.
  • Each branch runs only if its value matches.
  • default is used if no other case matches.
