Output 1

Outputting to the screen is done with the print command. 

print("Hello, World!")
Should result in the following output:
Hello, World!

Key points:

  • The print function is used to output text to the screen.
  • Text to be printed should be enclosed in quotes.
  • print is a function. Functions can take more than arguments by using a comma eg:
    print("Hello", "World!")

Output 2

print is a function. Functions can take more than one argument by using a comma 

print("Hello",2, "you.")
Should result in the following output:
Hello 2 you.

Passing multiple arguments negates the need for casting

Variables

Variables are used to store data in a program. 

variable_name = value
name = "Alice"
age = 30
global score = 100

Key points:

  • The = operator is used to assign a value to a variable.
  • Variable names cannot have spaces. You can use an underscore if the name has 2 or more words eg:
    shoe_size = 5
