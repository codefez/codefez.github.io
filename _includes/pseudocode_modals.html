Output 1

Outputting to the screen is done with the print command. 

print("Hello, World!")
Should result in the following output:
Hello, World!

Key points:

  • The print function is used to output text to the screen.
  • Text to be printed should be enclosed in quotes.
  • print is a function. Functions can take more than arguments by using a comma eg:
    print("Hello", "World!")

Output 2

print is a function. Functions can take more than one argument by using a comma 

print("Hello",2, "you.")
Should result in the following output:
Hello 2 you.

Passing multiple arguments negates the need for casting

Variables

Variables are used to store data in a program. 

variable_name = value
name = "Alice"
age = 30
global score = 100

Key points:

  • The = operator is used to assign a value to a variable.
  • Variable names cannot have spaces. You can use an underscore if the name has 2 or more words eg:
    shoe_size = 5
  • Variables can be reassigned to different values during the execution of the program.
  • a variable can be given global scope by using the global keyword.

Casting Variables

The datatype of a variable can be changed using casting

  • str(500) will cast (convert) the integer 500 to "500".
  • int("500") will cast (convert) the string "500" to the integer 500.
  • float("500.25") will cast (convert) the string "500.25" to the float 500.25.

Output 3

We can join several strings together using the + operator.
This is called concatenation.

print("Hello" + " " + "you.")
Should result in the following output:
Hello you.

We can actually concatenate variables as well.

name = "Alice"
print("Hello, " + name + "!")
Should result in the following output:
Hello, Alice!

Passing multiple arguments negates the need for casting

Casting Variables 2

We can join several strings together using the + operator.
This is called concatenation.

print("Hello" + " " + "you.")
Should result in the following output:
Hello you.

If we concatenate variables from different datatypes, we need to cast them to strings first.

name = "Alice"
age = 25
print(name + " is " + str(age) + ".")
Should result in the following output:
Alice is 25.
