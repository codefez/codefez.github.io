Variables

Variables are used to store data in a program.

variable_name = value
name = "Alice"
age = 30
global score = 100

Key points:

  • The = operator is used to assign a value to a variable.
  • Variable names cannot have spaces. You can use an underscore if the name has 2 or more words eg:
    shoe_size = 5
  • Variables can be reassigned to different values during the execution of the program.
  • a variable can be given global scope by using the global keyword.

Casting Variables

The datatype of a variable can be changed using casting

  • str(500) will cast (convert) the integer 500 to "500".
  • int("500") will cast (convert) the string "500" to the integer 500.
  • float("500.25") will cast (convert) the string "500.25" to the float 500.25.

Input and Output

We can use the input() function to get user input and the print() function to display output.

name = input("What is your name? ")
print("Hello, " + name + "!")

The above code will prompt the user to enter their name and then greet them with a personalized message. For example, if the user enters "Alice", the output will be:

Hello, Alice!

Input and Casting

We can cast the result of input() to a different datatype if needed.

number = int(input("Enter a number to work out the square: "))
print("The square of " + str(number) + " is " + str(number * number) + ".")

The above code will prompt the user to enter a number and then display its square. For example, if the user enters "5", the output will be:

The square of 5 is 25.

By default, the input() function returns a string, so we need to cast it to an integer using int() before performing the calculation.

String length

We can use the .length method to get the length of a string.

string = "codefez"
print(string.length)

The above code will output the length of the string "codefez", which is 7. For example, the output will be:

7

String Indexing

We can use indexing to access individual characters in a string. With pseudocode this is done by using the substring method.

string = "codefez"
print(string.substring(0))
print(string.substring(4,6))

The above code will output the first character of the string "codefez" and then the characters from index 4 to 5. For example, the output will be:

c
fez

for loops

We can use for loops to repeat a block of code a specific number of times.

This is known as a counter-controlled loop because we use it when we know the number of iterations in advance.

for i=1 to 5
    print(i)

The above code will repeat the print statement 5 times. For example, the output will be:

1
2
3
4
5

The variable i is called the loop counter. It is initialized to 1 and incremented by 1 each time the loop executes, taking on the values from 1 to 5 during the execution of the for loop.
It is also known as a local variable because it only exists within the scope of the loop. It cannot be accessed outside of the loop.

for loops

We can use a for loop to iterate over each character of a string.

string = "codefez"
for i=1 to string.length
    print(string[i])

The above code will repeat the print statement for each character in the string. For example, the output will be:

c
o
d
e
f
