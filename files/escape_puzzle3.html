Escape Room Puzzle 3: Another Truth

When we create logic diagrams or solve logic equations, we use truth tables to determine what the output is for a given set of inputs. The logic_solver function returns the output for a given truth table row based on logic gates given as an argument. The gates should be configured as follows: gate1 takes inputs A and B. gate2 takes as inputs the result of gate1 and input B. gate3 takes as inputs the result of gate2 and input C. If a 4th gate is given as an argument, it is a note gate which is placed before input C. The function should return the the final value for truth table labelled by `row` number given as an argument.
Please keep all solved puzzles a secret so that everyone can enjoy the challenge.
