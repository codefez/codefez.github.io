Quick reference

# read whole file
    with open('sample_text.txt', 'r') as f:
        contents = f.read()
                
# read lines and store in a list
with open('sample_text.txt') as f:
    lines = f.readlines()

Interactive tasks (Codapi)

Each task below is interactive. Edit the code in the editor and click Run.

Task 1 — Read the first line

Open sample_text.txt and print only the first line.

# Task 1: print the first line of sample_text.txt
with open('sample_text.txt', 'r') as f:
    first = f.readline()
    print(first.strip())

Task 2 — Count lines

Open the same file and print the number of lines it contains.

# Task 2: print number of lines in sample_text.txt
with open('sample_text.txt', 'r') as f:
    lines = f.readlines()
print(len(lines))
3

Task 3 — Write a new file

Create student_notes.txt and write three short notes (one per line). Print a confirmation message.

# Task 3: write three notes to student_notes.txt
with open('student_notes.txt', 'w') as f:
    f.write("Note 1\n")
    f.write("Note 2\n")
    f.write("Note 3\n")
print("Wrote student_notes.txt")
